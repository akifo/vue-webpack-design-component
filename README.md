# Vue.js Webpack design-component

[Live demo](http://akifo.github.io/vue-webpack-design-component/)

Built with [Vue.js](http://vuejs.org) 

## Building

``` bash
npm install
# watch:
npm run dev
# build:
npm run build
```

## start app
Start a dev server at `localhost:8080`:

``` bash
npm start
```
